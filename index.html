About This Project

This was inspired by @kosmari's https://glitch.com/edit/#!/offline-with-serviceworker, but uses sw-precache to generate an offline-first service worker that will automatically be kept up to date based on whatever's in your public/ folder, without you having to remember to update anything.

It also includes a cache-first runtime caching strategy for all assets that are loaded via Glitch's CDN.

Finally, it includes a web app manifest, meeting the requirements for a progressive web app.

Remix Me!

The project's source is at https://glitch.com/edit/#!/pwa-demo.

It should be everything you need to get started with your own progressive web app!
